End-to-End Insurance Analytics Project
Overview

This repository showcases an end-to-end Insurance Analytics project using Power BI, Tableau, Excel, and SQL. The project demonstrates the ability to analyze insurance data, generate actionable insights, and create interactive dashboards to support decision-making at both policy-level and branch-level.

Project Objective

Analyze insurance business data including invoices, opportunities, and meetings.

Provide insights at branch and policy levels to support management decisions.

Track key metrics like revenue, meetings, opportunity conversion rates, and policy trends.

Build interactive dashboards for performance monitoring.

Data Sources

The project uses multiple datasets:

Invoice Data: Revenue transactions, branch, account executive, policy details, and amount.

Meeting Data: Account executive meetings, attendance, and dates.

Opportunity Data: Opportunities, revenue amounts, stage, branch, and product group.

## Dashboards
1. Branch Dashboard

Tracks branch-level and account executive performance.

Key KPIs:

Total Revenue by Branch

Number of Meetings

Opportunity Conversion Rate

Revenue by Product Group

Target vs Achievement

Visualized using Power BI and Tableau.

Helps management identify top-performing branches and monitor overall business health.

2. Policy Dashboard

Focuses on policy-level insights including new vs renewal policies.

Key KPIs:

Number of Policies (New vs Renewal)

Policy Revenue Trends

Policy Conversion Rates

Average Revenue per Policy

Allows tracking of policy trends and revenue contribution at a granular level.

Built with Power BI and Excel for detailed analysis.

Tools Used

SQL: Data extraction, transformation, and aggregation.

Excel: Pivot tables, charts, and detailed analysis.

Power BI: Interactive dashboards for branch and policy KPIs.

Tableau: Visual analytics for business insights.
